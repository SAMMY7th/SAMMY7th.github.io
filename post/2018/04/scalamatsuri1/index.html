OSSに初貢献した話 - ScalaMatsuri2018レポートその1 #ScalaMatsuri

2018/04/13   #IT勉強会  #Scala 

3/16〜3/18 3日間に渡って開催された ScalaMatsuri2018 に参加してきました。 すっかり遅くなってしまったのですが、感想ブログを書いていこうと思います。

3⁄16 1日目 OSSハッカソン

1日目のトレーニングDayでは OSSハッカソン に参加しました。

私はScalikeJDBC, Skinny Frameworkの瀬良さん( @seratch, @seratch_ja ) のチームに参加しました。 ScalikeJDBCを利用したことがあるのと、瀬良さんにはいつもScala関西Summitでハンズオンを開催いただいたりご登壇いただいたりと大変お世話になっているので、少しでも貢献できたら・・・と思い、第1希望にしていました。

他の参加者の方々との軽い自己紹介のあと早速作業を開始することに。

私はOSSにPRを出したことがなく、最初「OSS PR マナー」でググりながら「す、すみませんそもそもissue着手するときって勝手にやっていいもんなんですか(震え声)」みたいな質問をするという状態でした。(※ よいそうです。)

OSSにPR出したことなくて、そもそもの出し方からおそわってるなう #ScalaMatsuri

— SAMMY(きの子) (@aa7th) 2018年3月16日

そして「これ多分簡単ですよ」とおすすめされたissueに着手。瀬良さんの超手厚いサポートのお陰で、2つPRを出すことに成功！無事マージもされました！！

3rd pull requests to me at #ScalaMatsuri OSS hack-a-thon has been merged. Thanks .@aa7th! https://t.co/vcfZMoGY0i

— Kazuhiro Sera (@seratch) 2018年3月16日

ScalikeJDBCのコントリビューターになりました 🙌

https://t.co/F9wsLKw9lw

「2つPR出してマージされた」と言ったら色んな人に「すごい！！！！！！」とと褒めて頂けたんですが、事前に瀬良さんが簡単なissueを用意してくれていた上に、自己解決できずにほぼほぼ瀬良さんに聞きながらやったので、なんかもう褒められると申し訳ない気持ちになるレベル・・・。おいしいところだけもってっちゃってすみませんすみません。

参加して本当によかった！

参加してまじでよかったという感想しかありません。

  • メンテナの方のサポートを受けながらとはいえ、PRを出す経験をすることにより、OSSへPRを出すことの心理的障壁が大分下がった。
  • 質問したときにいただいたアドバイス(sbtプラグイン情報などなど)が勉強になった。
  • 瀬良さんの作業の速さをみて、私ももっと速度とか効率とかをあげていかないといけないよな・・・って思った。(いい刺激になった)

いい機会をいただけましたし、1日目の時点で「大阪からはるばる来た甲斐あったわぁ・・・」ってなりました。

関係者のみなさま、本当にありがとうございました。

ところで！実は！！4/21(土) その瀬良さんが大阪にいらっしゃいます！！！

4⁄21(土) ScalaMatsuri 2018 Osakaサテライト

ScalaMatsuri 2018 Osakaサテライト - connpass https://connpass.com/event/82566/

ScalaMatsuri 2018 プログラムの中から 瀬良さん、粕谷さん ( @daiksy ) に セッションをご再演いただきます。また、数本のセッション録画動画を視聴する予定です。 場所は新大阪のMOTEX株式会社さん。

終了後は ビアバッシュ も開催予定です。(申し込みページが別なのでご注意を)

OSS貢献に興味がある方、OSSだけじゃなくてとにかくScalaの話をしたいぜ聞きたいぜという方、ぜひぜひ4/21イベントお越し下さいませ。お待ちしております！
